MULTISENSOR FUSION ARCHITECTURE
─────────────────────────────────────────────────────────────────────────────
  Weights:     Sentinel-1 (45%) + Sentinel-2 (15%) + DEM (15%) + 
               Rainfall (15%) + GroundSourceIncidents (10%)
  
  Sentinel-1 (45%):
    - scene_support_score = clamp(baseline_scene_count / 6.0)
    - change_intensity_score = clamp(|change_db| / |threshold_db|)
    - runtime_signal_score = average(scene_support, intensity)
  
  Sentinel-2 (15%): PLACEHOLDER — No live data
    - water_presence_score from NDWI/MNDWI (metadata only)
    - cloud_free_fraction = 1.0 - (cloud_cover_pct / 100)
  
  DEM, Rainfall, Incidents (40%): SCAFFOLDING — No live data
  
  Confidence Label Mapping:
    score >= 0.85  →  VERY_HIGH
    score >= 0.70  →  HIGH
    score >= 0.50  →  MEDIUM
    score >= 0.30  →  LOW
    else           →  VERY_LOW

TEMPORAL FRAMES
─────────────────────────────────────────────────────────────────────────────
  Baseline:     window=metadata.baseline_window (e.g., "2025-06-01 to 2025-09-30")
                Used for Sentinel-1 baseline median image
  
  Event:        window=metadata.event_window (e.g., "2026-01-01 to 2026-02-28")
                Used for Sentinel-1 event median image
  
  Change:       DERIVED: baseline_start → event_end
                Used for flood product (change = event - baseline)
  
  Progression:  Alternative event_window candidates from metadata
                (if event_window_candidates array present)

DEPLOYMENT
─────────────────────────────────────────────────────────────────────────────
  To Enable Live EE:
    1. Set environment variable: HAWKEYE_ENABLE_EE_LIVE_TILES=true
    2. Ensure earthengine CLI authenticated: earthengine authenticate
    3. Verify GCP_PROJECT_ID set in .env
    4. Verify service account has EE API access
  
  Default (Recommended for Initial Deploy):
    • Leave HAWKEYE_ENABLE_EE_LIVE_TILES unset
    • System operates in offline mode (100% reliable)
    • Placeholder tiles render as gray underlay
    • All functionality available except live MapID generation
