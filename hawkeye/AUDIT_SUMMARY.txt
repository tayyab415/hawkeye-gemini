 KEY FINDINGS

1. CURRENT STATE: HYBRID (Real Infrastructure + Static Geometry)
   
   LIVE COMPONENTS:
   ✓ BigQuery infrastructure queries (get_infrastructure_at_risk)
   ✓ BigQuery historical flood patterns (query_historical_floods)
   ✓ WebSocket bidirectional agent communication
   
   STATIC/MOCK COMPONENTS:
   ✗ Flood extent geometry (loaded from file: flood_extent.geojson)
   ✗ Growth rate = HARDCODED 12% per hour (see line 102 of earth_engine_service.py)
   ✗ Population density = HARDCODED 15,000/km² (see line 14 of earth_engine_service.py)
   ✗ Provenance metadata (loaded from analysis_provenance.json)
   ✗ No live raster/tile delivery (only static GeoJSON vectors)

2. CRITICAL GAPS

   Gap 1: No Live Raster/Tile Delivery
   - Missing: SAR backscatter, NDWI water body, change detection heatmaps
   - No TMS/XYZ tile endpoint for Cesium

   Gap 2: Missing Provenance/Confidence Metadata
   - No per-pixel confidence scores
   - No scene acquisition details or quality metrics
   - No data freshness indicators

   Gap 3: No Live EE Computation API
   - EarthEngineService only loads files, doesn't compute
   - No parameter override capability (e.g., different threshold_db)
   - No async task tracking

   Gap 4: Incomplete WebSocket Contract
   - ee_update event lacks structure; no raster references
   - No explicit confidence breakdown

   Gap 5: Frontend Tile Layer Management Missing
   - No raster opacity/blend controls
   - No layer composition (SAR + change + water)

3. PHASE-1 DELIVERABLE: CONCRETE CONTRACT

   Backend:
   - POST /api/earth-engine/analyze → Submit analysis, return task_id (202)
   - GET /api/earth-engine/tasks/{task_id} → Poll status + progress
   - GET /api/earth-engine/tasks/{task_id}/result → Fetch final result
   - GET /api/earth-engine/tiles/{task_id}/{layer}/{z}/{x}/{y}.png → Raster tiles

   WebSocket:
   - ee_analysis_start: { task_id, estimated_time_seconds }
   - ee_analysis_update: { task_id, phase, progress_pct, message }
   - ee_analysis_complete: { task_id, analysis_provenance, geometry, tiles, metrics }

   Frontend:
   - Add "Run Live Analysis" button to EarthEnginePanel
   - Display progress bar with phase updates
   - Render raster layers with opacity control
   - Update provenance panel with task metadata
